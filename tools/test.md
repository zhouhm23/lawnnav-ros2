## 测试脚本使用说明

### test1_slam_nav_test.py — SLAM与导航综合测试

**统一测试入口**，替代旧版 `goal1-4.py`、`goal_all.py`、`test2`、`test3`。

```bash
# 仅闭合路径相对位姿误差 (RPE) — 导航 + 逐点人工输入地面真值
python3 tools/test1_slam_nav_test.py --mode rpe

# 仅静态定位稳定性 — 60s 静止记录 5Hz 采样
python3 tools/test1_slam_nav_test.py --mode static

# 全部测试（先 RPE 后 static）
python3 tools/test1_slam_nav_test.py --mode all
```

**坐标系转换**：用户在终端以**论文坐标系** (y⁺=车头, x⁺=右侧, 起点1=(0.4,0)) 输入地面真值，脚本自动转换为 Nav2 map 坐标系进行 RPE 计算。RPE 使用相对位移差，不受全局坐标不对齐影响。

**输出**：
| CSV 文件 | 内容 |
|----------|------|
| `tools/rpe_results.csv` | 逐段 RPE (4段) + 端到端闭合误差，含 SLAM/GT 位姿对照 |
| `tools/static_stability_results.csv` | RMSE 抖动 + 最大漂移量（位置 & 航向） |
| `tools/pose_log_*.csv` | 60s 原始位姿采样数据 |

**已废弃的旧脚本**：`goal1.py` ~ `goal4.py`、`goal_all.py`、`test2_measure_trajectory_error.py`、`test3_measure_map_quality.py`

**地图尺寸精度测试**改为全人工流程（人控运动 + 人工测量 + 人工记录），参考工具见 `test4_measure_map_accuracy.py`。
