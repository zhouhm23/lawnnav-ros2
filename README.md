# 基于rtabmap的单深度相机+边缘外扩型细胞分解算法的全覆盖程序
本项目旨在设计一种针对复杂环境区域自动全覆盖的程序。目前市面上自主导航机器人大多使用单雷达（slam）或雷达+视觉（RTAB-VSLAM）方案。然而前者只能2D建图，面对低矮障碍物会一直碰撞，后者传感器多效果最好但价格贵，因此本项目介绍了一种基于rtabmap的单相机建图和定位方案，具有低成本，能3D建图导航；另外本项目还在一般细胞分解算法上增加了边缘外扩功能以提高覆盖率，已达到传统方案水准的同时大幅降低成本。
